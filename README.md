# 💊 Structural Bioinformatics & In Silico Drug Design

### 🎓 Project Context
While transcriptomics and genomics identify target genes, structural bioinformatics allows us to understand the physical proteins those genes produce. This repository serves as a digital protocol book detailing the *in silico* workflows I utilized during my academic research to visualize macromolecular structures, predict protein folding, and simulate drug-target interactions.

### 🎯 Objective
To provide reproducible Standard Operating Procedures (SOPs) for 3D protein visualization, homology modeling, and molecular docking simulations using industry-standard GUI software and web servers.

### 🛠️ Core Technologies & Software
* **Database Mining:** NCBI Entrez, PubMed, PDB (Protein Data Bank), UniProt
* **3D Visualization:** PyMOL, RasMol, Discovery Studio
* **Homology Modeling:** Swiss-Model, Phyre2
* **Molecular Docking:** ArgusLab, AutoDock

### 📂 Standard Operating Procedures (SOPs) Included
1. **[Literature & Database Mining](01_Literature_and_Database_Mining.md):** Efficiently retrieving academic literature and protein sequences using Entrez and Mendeley.
2. **[3D Protein Visualization](02_3D_Protein_Visualization.md):** Fetching X-ray crystallography data (PDB) and visualizing secondary/tertiary structures.
3. **[Homology Modeling](03_Homology_Modeling_and_Validation.md):** Predicting 3D protein structures from primary amino acid sequences and validating them via Ramachandran plots.
4. **[Molecular Docking Pipeline](04_Molecular_Docking_Pipeline.md):** Preparing active sites and simulating ligand binding affinities (e.g., targeting HIV-1 Protease).
